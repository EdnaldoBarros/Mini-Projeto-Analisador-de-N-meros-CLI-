Analisador de Números

Projeto simples em Python que analisa uma lista de números fornecida pelo usuário via terminal.

Funcionalidades

- ✔️ Identifica o maior número
- ✔️ Identifica o menor número
- ✔️ Calcula a média
- ✔️ Mostra a quantidade de números
- ✔️ Tratamento de erros
- ✔️ Estrutura modularizada


Tecnologias Utilizadas

Python 3.x
